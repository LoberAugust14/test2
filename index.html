1. First Create a profile on Github.com
  2. Next create a repository on Github specific to your desired website.
  3. After creating a repository, it is time to begin coding.
  4. Using Sublime Text, create an html that will eventually become your website.
  5. Inside this html you can change the background color, font type, font color and much more.
  6. After your html is to your liking, you must commit your html file to your git repository using Terminal.
  7. Then once your files are loaded to your Github repository, go to Netlify.com and create an account linked to your Github account.
  8. In Netlify, you will be able to create your website using your Github repository.
  9. Lastly, Netlify will give you random domain name which will be clickable, from their you can view your website.
